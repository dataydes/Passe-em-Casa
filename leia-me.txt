Intalação do servidor passo a passo:
1 - Instalar o Node.js no site do Node
2 - Executar o comando  para instalar a estrutura do react native: npm install -g create-react-native-app
3 - Cria a aplicação HelloWord: create-react-native-app HelloWorld
4 - npm run web
